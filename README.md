# Блок "Progress" - тестовое задание Ozon Банк.

Демонстрационное приложение - https://nozemiso.github.io/ozon-frontend-test/

## Описание

Блок Progress - блок, предназначенный для отображения прогресса выполнения процессов в мобильных web-приложениях.
Прототип блока разработан с использованием HTML, CSS, JS без использования каких-либо дополнительных библиотек/фреймворков/надстроек.

## Api блока

Класс `Progress`, реализующий монтирование и функционал блока имеет следующий api:

* `constructor(targetElement: HTMLElement)` - монтирует блок Progress в требуемый html-элемент
* `setValue(value: int, min=0: int, max=100: int)` - устанавливает текущее положение блока, опционально можно настроить диапазон значений
* `setHidden(state: bool)` - устанавливает видимость блока: True - скрыть, False - отобразить
* `setAnimated(state: bool)` - управляет анимацией вращения блока: True - запустить, False - остановить

Блок разработан под легкое переиспользование в составе других приложений и не содержит прямых зависимостей от файлов приложения. Он принимает размер родительского блока, цвета задаются переменными отдельно для блока Progress.

## Итоговое приложение

Для демонстрации работы, а также в качестве примера использования блока разработано web-приложение.
Оно также разработано на чистых HTML, CSS, JS и позволяет отдельно управлять каждым из параметров блока Progress.
Приложение адаптируется под ориентацию и размеры экрана засчет определения ориентации и вёрстки с использованием Flexbox и Grid.
